Automate Your Video Workflow Securely

Autotube is a lightweight, open-source pipeline that cleans raw datasets, renders stunning data visualizations, and schedules video uploads directly to YouTube.

View Source Code

100% Local Processing

Autotube runs entirely on your own local environment. Your OAuth credentials, YouTube client secrets, and refresh tokens never leave your machine.

Dynamic Rendering

Uses FFmpeg and custom charting engines to render high-framerate Shorts and long-form data animations from raw CSV datasets automatically.

Secure OAuth Integration

Authenticates directly with Google OAuth2 APIs using the standard authorization code grant flow to request write-only YouTube video upload permissions.
