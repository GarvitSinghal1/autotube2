# Autotube Web Pages

These pages are generated to help you pass the Google OAuth consent verification or fill out the required fields.

## Files
* `index.html` — Application Home Page
* `privacy.html` — Privacy Policy Link
* `terms.html` — Terms of Service Link
* `style.css` — Modern dark styling
* `logo.png` — App Logo

## How to Deploy to GitHub Pages (Free)

The easiest way to get public HTTPS links is by using **GitHub Pages** on your existing repository:

1. On GitHub, go to your repository **Settings** > **Pages** (under the "Code and automation" section).
2. Under **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: Select **main**
   - Folder: Select **/docs** (instead of `/ (root)`)
3. Click **Save**.
4. After a minute, GitHub will activate the site at:
   `https://garvitsinghal1.github.io/autotube2/`
