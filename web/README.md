# Autotube Web Pages

These pages are generated to help you pass the Google OAuth consent verification or fill out the required fields.

## Files
* `index.html` — Application Home Page
* `privacy.html` — Privacy Policy Link
* `terms.html` — Terms of Service Link
* `style.css` — Modern dark styling
* `logo.png` — App Logo

## How to Deploy to GitHub Pages (Free)

The easiest way to get public HTTPS links is by using **GitHub Pages**:

1. Create a new public repository on GitHub named `autotube` (or any name you prefer).
2. Upload the contents of this `web/` folder directly to the root of that repository.
3. On GitHub, go to your repository **Settings** > **Pages** (under the Code and automation section).
4. Under **Build and deployment**, select **Deploy from a branch** and set the branch to `main` (or `master`) and folder to `/ (root)`. Click **Save**.
5. After a minute, GitHub will give you a public URL (e.g., `https://<your-username>.github.io/autotube/`).

### OAuth Consent Screen Mapping:
